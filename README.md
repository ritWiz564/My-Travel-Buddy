My Travel Buddy is a full-stack web application designed to simplify trip planning and itinerary management. The platform allows users to create, organize, and manage travel plans in a structured manner.

Features

Trip creation and management

Itinerary organization

Backend API integration

Persistent data storage

Tech Stack

Frontend: React.js, HTML, CSS
Backend: Node.js, Express.js
Database: MongoDB
Version Control: Git, GitHub

Learning Outcome

This project demonstrates practical implementation of client-server architecture, RESTful APIs, database design, and collaborative development workflows.

Future Scope

The system can be extended with AI-based travel recommendations and real-time collaboration features.
